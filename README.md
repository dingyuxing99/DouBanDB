# Douban Database:
本实习指导书适用于计算机专业、网络工程和地理信息专业。通过编制实际的、完整的数据库系统应用，让学生掌握数据库系统设计的基本原理和方法。具体包括：系统的需求分析、数据库概念设计、数据库的逻辑设计、关系数据库理论、数据持久层设计和SQL语言使用等。
